📚 Arrays in JavaScript - Part 2: Adding & Removing Elements

📌 Topics Covered

  • Adding elements: push(), unshift(), splice()
  • Removing elements: pop(), shift(), splice()
  • Queue vs Stack operations
  • Performance considerations
💡 Quick Reference:
• push() → Add to END
• pop() → Remove from END (FASTEST)
• unshift() → Add to BEGINNING
• shift() → Remove from BEGINNING (SLOWEST)
• splice() → Add/Remove anywhere
