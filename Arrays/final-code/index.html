📚 Arrays in JavaScript - Complete Guide

📌 What is an Array?

An array is a special variable that can hold multiple values (elements) in a single variable. Arrays are ordered lists that can store any data type (numbers, strings, objects, even other arrays).

📌 Why Use Arrays?

  • Store multiple values → Instead of 100 variables, use 1 array
  • Ordered data → Maintain sequence of items
  • Easy iteration → Loop through all elements
  • Built-in methods → Powerful operations (map, filter, reduce)

📌 Array Characteristics

  • Zero-indexed → First element is at index 0
  • Dynamic size → Can grow or shrink automatically
  • Mixed types → Can hold different data types
  • Reference type → Arrays are objects in JavaScript

📌 Creating Arrays

// Array literal (preferred)
const fruits = ['apple', 'banana', 'orange'];

// Array constructor
const numbers = new Array(1, 2, 3);

// Empty array
const empty = [];

// Array with mixed types
const mixed = [1, 'hello', true, null, {name: 'John'}];

📌 Common Array Methods

  • push() / pop() → Add/remove from end
  • unshift() / shift() → Add/remove from beginning
  • map() → Transform each element
  • filter() → Filter elements based on condition
  • reduce() → Reduce to single value
  • forEach() → Iterate through array
  • find() / findIndex() → Find element/index
  • some() / every() → Test conditions
  • sort() → Sort elements
  • slice() / splice() → Extract/Modify portions
