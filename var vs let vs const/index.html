var vs let vs const

📌 What are Variables?

Variables are containers used to store data values in JavaScript.

📌 Types of Variable Declarations

  • var → old way (before ES6)
  • let → modern, block-scoped
  • const → constant, cannot be reassigned

📊 Comparison Table

Feature var let const
Scope Function Block Block
Hoisting Yes (undefined) Yes (TDZ) Yes (TDZ)
Reassignment Allowed Allowed ❌ Not allowed
Redeclaration Allowed ❌ Not allowed ❌ Not allowed

📌 Key Differences

1. Scope

var is function-scoped, while let and const are block-scoped.

2. Hoisting

var is initialized as undefined, but let and const are in Temporal Dead Zone (TDZ).

3. Reassignment

const cannot be reassigned after declaration.

📌 Best Practice

  • Use const by default
  • Use let if value needs to change
  • Avoid var in modern JavaScript
