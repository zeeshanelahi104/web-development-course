Hoisting in JavaScript

📌 What is Hoisting?

Hoisting is JavaScript's behavior of moving declarations to the top of their scope during the creation phase.

📌 What gets Hoisted?

  • var → hoisted as undefined
  • function declarations → fully hoisted
  • let & const → hoisted but in Temporal Dead Zone (TDZ)

📌 Example Flow

console.log(a); // undefined
var a = 10;

hello(); // works

function hello() {
  console.log("Hello");
}

📌 Temporal Dead Zone (TDZ)

Variables declared with let and const cannot be accessed before initialization.
