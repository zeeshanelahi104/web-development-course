Call Stack in JavaScript

📌 What is Call Stack?

Call Stack is a data structure used by JavaScript to keep track of function execution. It follows LIFO (Last In, First Out).

📌 How it Works

  1. Global Execution Context is pushed first
  2. When a function is called → pushed into stack
  3. When function finishes → popped out

📌 Key Points

  • Single-threaded → only one task at a time
  • Stack overflow happens when too many calls
  • Works synchronously
